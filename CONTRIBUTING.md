# Contributing to NeoStation Website

Thank you for your interest in contributing to the NeoStation official website! We welcome improvements to content, design, performance, and accessibility.

## How to Contribute

### Reporting Issues

- Check if the issue already exists in the [issue tracker](https://github.com/misobadev/neostation-web/issues).
- Provide a clear description, steps to reproduce (if applicable), and screenshots when relevant.

### Submitting Changes

1. **Fork** the repository.
2. **Create a branch** from `main` with a descriptive name (`fix/typo-homepage`, `feat/dark-mode-toggle`, etc.).
3. **Make your changes** following the existing code style.
4. **Test locally** with `npm run build` to ensure the build passes.
5. **Submit a Pull Request** with a clear description of what changed and why.

## Code Style

- We use [Biome](https://biomejs.dev/) for linting and formatting.
- Run `npm run lint` and `npm run format` before committing.
- Follow the existing Astro and Tailwind CSS patterns.

## Questions?

Feel free to open a [Discussion](https://github.com/misobadev/neostation-web/discussions) or reach out on our [Discord](https://discord.gg/xE2kgKsRVq).
